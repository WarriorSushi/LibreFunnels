# Naming

## Final Name
Product name: LibreFunnels

Working plugin title: LibreFunnels for WooCommerce

Preferred plugin slug: `librefunnels`

## Why This Name
LibreFunnels communicates software freedom more clearly than just "free." It supports the project's open-source promise while leaving room for ethical future monetization through support, hosting, templates, services, marketplace revenue, gateway adapters, or managed analytics.

The name avoids the two main collision risks discovered during planning:
- `OpenFunnel` is already used by a YC-backed GTM intelligence company.
- `OpenCart` is an existing ecommerce platform, making `OpenCarts` risky and confusing.

## Positioning
Primary tagline:

Free WooCommerce funnels, checkout, upsells, downsells, and order bumps.

Positioning line:

The open-source funnel builder WooCommerce users deserved.
