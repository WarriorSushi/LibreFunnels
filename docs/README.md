# LibreFunnels Project Docs

LibreFunnels is a 100% free, open-source WooCommerce funnel builder intended to become a full-featured alternative to CartFlows.

The plugin must support physical products, digital products, services, subscriptions where feasible, order bumps, upsells, downsells, cross-sells, checkout replacement, funnel routing, analytics, A/B testing, templates, and a visual canvas builder.
